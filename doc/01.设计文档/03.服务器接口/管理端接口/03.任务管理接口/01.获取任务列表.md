# 概述

1. 获取任务列表
2. 任务列表默认按时间降序排列(后创建的排在前面)

# 列表

1. 接口名: /admin/task/list

2. 接口参数

>>必填
* token: 管理员登录后获取的token

>>可选
* type: 任务类型，默认为全部任务(=0)
* sort: 排序方式
```json
{
    "order": "DESC" | "ASC",
    "field": "create" | "username" | ...
}
```
* page: 当前页
* size: 一页最多显示的用户数量
* lang: 客户端语言，默认中文('zh')

3. 返回值
* 成功:
```json
{
}
```

* 失败
```json
[
    "ERR.NULL_PARAM",// 参数为空
]
```
