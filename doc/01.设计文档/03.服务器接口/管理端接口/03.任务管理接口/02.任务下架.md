# 概述

1. 下架任务(批量，如果只需要下架一个，则数组填一个taskid即可)
2. 下架后的任务不可在大厅中看到，不可领取

# 列表

1. 接口名: /admin/task/offShelf

2. 接口参数

>>必填
* token: 管理员登录后获取的token
* taskids[]: 需要下架的任务ID数组

>>可选
* lang: 客户端语言，默认中文('zh')

3. 返回值
* 成功:
```json
{
}
```

* 失败
```json
[
    "ERR.NULL_PARAM",// 参数为空
    "ERR.TASK_NOT_EXISTED",// 下架任务不存在
]
```
