# 概述

1. 解冻用户(批量，如果只需要冻结一个，则数组填一个uid即可)
2. 解冻用户的前一个状态需要是FROZEN

# 列表

1. 接口名: /admin/user/unfreeze

2. 接口参数

>>必填
* token: 管理员登录后获取的token
* uids[]: 需要冻结的用户ID数组

>>可选
* lang: 客户端语言，默认中文('zh')

3. 返回值
* 成功:
```json
{
}
```

* 失败
```json
[
    "ERR.NULL_PARAM",// 参数为空
    "ERR.USER_NOT_EXISTED",// 解冻用户不存在
]
```
