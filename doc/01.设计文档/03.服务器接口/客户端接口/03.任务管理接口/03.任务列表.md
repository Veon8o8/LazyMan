1. 接口名: /client/address/del

2. 接口参数

>>必填
* token: 成功登录返回的令牌(token)
* type: 任务类型(0.所有;1.代购物品;2.代取快递)

>>可选
* page: 当前页
* size: 一页最多显示的任务数量
* lang: 客户端语言，默认中文('zh')

3. 返回值
* 成功:
```json
```

* 失败
```json
[
    "ERR.NULL_PARAM",// 参数为空
    "WRONG_TASK_TYPE",// 任务类型错误
]
```
